
# 📊 Student Performance Indicator Analysis

## 📌 Project Overview
This project focuses on identifying the key factors that influence student performance in exams. By analyzing a dataset of 1,000 students, I explored how variables like gender, ethnicity, parental education, and lunch types impact scores in Math, Reading, and Writing.

As a **Computer Science student**, I developed this project to demonstrate my skills in **Exploratory Data Analysis (EDA)**, data visualization, and building predictive models using Python's data science ecosystem.

## 🚀 Key Features
- **Comprehensive EDA:** Analyzed distributions and correlations using Heatmaps, Violin plots, and Histograms.
- **Data Preprocessing:** Cleaned missing values, handled duplicates, and performed feature engineering (Total Score & Average).
- **In-depth Insights:** Documented over 15+ specific findings regarding student demographics and academic outcomes.
- **Predictive Modeling:** Built a regression pipeline to estimate student grades based on socio-economic features.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn
- **Environment:** Jupyter Notebook / Google Colab

## 📈 Major Insights
Based on the analysis in the notebook:
- **Gender Trends:** Female students generally outperformed male students, especially in Reading and Writing.
- **Nutrition Factor:** Students with "Standard" lunch consistently scored higher (approx. 10-12% more) than those with "Free/Reduced" lunch.
- **Preparation Impact:** Completing a test preparation course significantly reduced the variance in scores and boosted the overall average.
- **Parental Background:** Higher parental education levels (Associate's/Bachelor's/Master's) correlate with more consistent student performance.

## 📂 File Structure
- `student-performance.ipynb`: Full analysis, visualization, and documentation.
- `StudentsPerformance.csv`: Dataset used for the analysis.

## 🏁 How to Use
1. Clone the repository.
2. Open `student-performance.ipynb` in Google Colab or VS Code.
3. Ensure you have the `StudentsPerformance.csv` in the same directory.
4. Run all cells to see the visualizations and model metrics.
